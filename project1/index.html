Battle Run

Battle run is a beat'em played in a browser. The main premise of battle run is that your character will not stop running. The only control the player has in battle run is attacking and jumping. Battle run is a level based and played with a keyboard. Battle run does not have any story beyond running guy punches bad guys. The game has a cartoonish graphical style that takes inpiration from Viewtiful Joe.
